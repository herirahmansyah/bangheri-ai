# Skills

## push sekarang
Jalankan `git push -u origin main` untuk push kode ke GitHub. Jika autentikasi gagal:
1. Install GitHub CLI: download binary dari https://github.com/cli/cli/releases
2. Login: `gh auth login`
3. Update remote URL: `git remote set-url origin https://<username>:<token>@github.com/<username>/<repo>.git`
4. Push: `git push -u origin main`
